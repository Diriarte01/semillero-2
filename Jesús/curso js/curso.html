Ambito Global: tipos de variables: globales y locales globales: se puede usar en cualquier lugar del programa porque está definida en el programa principal y no dentro de una funcion. local: son variables que definimos en una funcion y que solo se deben usar en esa funcion Retornar un valor: las funciones pueden retornar un valor ademas de recibir valores. Permanece en una fila (cola). ejemplo de variacion de una cola. ejemplo: definir una funcion proximoEnLaFila, debe tomar un arreglo y va tomar un numero llamado elemento, se debe agregar el numero al final del arreglo y luego el primer elemento debe ser eliminado. la funcion proximoEnLaFila debe retornar el valor eliminado. function proximoEnLaFila(arreglo, elemento){ arreglo.push(elemento); // argregar al final del arreglo return arreglo.shift(); // Remover el primer elemento } var miArreglo = [1, 2, 3, 4, 5]; console.log("Antes: " + JSON.stringify(miArreglo)); console.log(proximoEnLaFila(miArreglo, 6)); console.log("Después: " + JSON.stringify(miArreglo)); JSON.stringify() permite mostrar el arreglo en una forma mas presentable en la consola.(cadena de caracteres en un formato especifico) Valores Booleanos: son verdadero (true) o falso(false) operadores de igualdad operador de igualdad (==) operador de igualdad estricta (===) Operadores de desigualdad operador de desigualdad (!=) operador de desigualdar estricta (!==) Operador mayor que (>) Operadores logicos AND (&&) OR (||) NOT (!) el signo de admiracion se coloca antes de la expresión a cambiar de valor por ejemplo si es X es true entonces !X seria false y viceversa.
